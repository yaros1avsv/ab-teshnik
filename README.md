# A/B Тест Помощник

Бот-помощник для A/B-тестов, понятный любому сотруднику.

## Идея

Для любого сотрудника: пишет обычным языком («хочу проверить новый заголовок»). Бот задаёт простые вопросы по одному, рекомендует метрику, считает объём и срок, объясняет без p-value и «выборки в абстракции» — в духе «~N недель при вашем трафике, увидим рост от X%».

После планирования создаёт эксперимент в PostHog (API): feature flag, группы, метрика. Простые изменения (текст, цвет) — через визуальный редактор PostHog; сложные — готовый короткий сниппет, деву остаётся вставить и задеплоить.

Пока тест идёт — роль контролёра: не подглядывать в шум, показывать прогресс набора, объяснять зачем.

По набору объёма — значимость и вердикт простым языком; при победе — команда на раскатку победителя.

## Запуск

```bash
./ab-test.sh
```
